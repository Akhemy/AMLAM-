## Descripción del proyecto

Sitio web institucional de una página (one page) para empresa de servicios atmosféricos en Mar del Plata. El objetivo principal es generar contacto directo por teléfono y WhatsApp, transmitir confianza y profesionalismo, y posicionarse localmente en Google.

---

## Stack técnico

- HTML5 semántico
- CSS3 (mobile-first, sin frameworks)
- JavaScript vanilla
- Sin dependencias externas de JS
- Fuentes: Inter via Google Fonts
- Deploy: Netlify
- Repositorio: GitHub

## Secciones del sitio

1. **Header** — sticky, logo, navegación desktop y hamburguesa mobile
2. **Hero** — foto camión, overlay, botones llamar y WhatsApp
3. **Tira habilitaciones** — ticker animado con certificaciones de la empresa
4. **Tira zonas** — ticker animado con zonas de cobertura
5. **Stats** — 500+ trabajos, 30+ años de experiencia, zonas de cobertura
6. **Servicios** — 7 tarjetas: pozos ciegos, destapes, cámaras sépticas, efluentes, servicio industrial, pozos de ascensores, biodigestores
7. **Sección asesoramiento** — galería de fotos + texto con equipos (camión atmosférico, hidrojet, desobstructores)
8. **Zonas de cobertura** — chips + carrusel automático de fotos por zona
9. **Por qué elegirnos** — 5 diferenciales con foto de camión de fondo
10. **Testimonios** — carrusel de reseñas de clientes
11. **CTA final** — llamada a la acción con teléfono y WhatsApp
12. **Footer** — contacto, medios de pago, secciones, crédito Creadoras3D

6. Deploy en Netlify con repositorio GitHub del cliente
7. Entrega de todas las cuentas al cliente
